## 使用 

推荐使用更快更合理的 `pnpm` 包管理器  [安装教程](https://pnpm.io/zh/installation)

1. 安装依赖

```shell
pnpm install
```

2. 开发

```shell
pnpm dev
```

3. 打包

```shell
pnpm build
```
